1. Регистрация:
    - зарегистрироваться может только Client
    - регистрация Coach только через админку
    - после регистрации пользователь сразу авторизуется

2. Авторизация:
    - Пользователь вводит или email или телефон
    - Пользователь вводит пароль
    - После успешного ввода кредов клиент попадает на профиль

3. Бронирование зала:
    - Клиент выбирает зал
    - Клиент выбирает время (свободное время получаем из зала)
    - Клиент подтверждает зал и время, списывается 1 посещение.
    - Клиенту предлагает забронировать тренера

4. Бронирование тренера:
    (Снизу должно быть подписано, что тренера можно забронировать, если забронирован слот в зале)
    - Клиент выбирает зал
    - Клиент выбирает тренера
    - Клиент выбирает день
    - Клиент получает расписание тренера и доступные слоты (слот забранированного времени в зале должен подсвечиваться)

5. Оплата абонементов:
    - Клиент получает список абонементов
    - Клиент выбирает абонемент
    - Клиент оплачивает
    - Клиент получает сообщение об успехе или провале операции
