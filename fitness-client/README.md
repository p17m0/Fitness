# Прототип 3: Брутализм (React + Tailwind)

Главная страница сайта HHsportFit в бруталистичном стиле с использованием чистого Tailwind CSS.

## Особенности дизайна
- Бруталистичный подход
- Агрессивные тени и толстые границы
- Моноширинный шрифт (Courier New)
- Яркие цвета и контрасты
- Грубая эстетика с акцентом на силу и мощь

## Запуск проекта

```bash
cd prototype-3-brutal
npm install
npm run dev
```

## Структура
- `/src/App.tsx` - основной компонент
- `/src/index.css` - базовые стили и кастомные классы
- Tailwind CSS с кастомными утилитами
